-------------------------------------
README for MC's Online Store
-------------------------------------

Purpose:
-------------------------------------
MC's Online Store is a simple web-based platform that allows users to browse and purchase various gadgets online. The store showcases a 
selection of products along with their descriptions and prices in South African Rand (ZAR). Users can add items to their cart and proceed to 
checkout.

Technologies Used:
-------------------------------------
This project is built using HTML and CSS for the structure and styling of the web page. It does not incorporate server-side scripting 
or databases, as it is a basic static website.

Viewing the Website Locally:
-------------------------------------
To view MC's Online Store on your local machine, follow these steps:

1. Download the project files (HTML, CSS, images, etc.) to your local computer.
2. Open the project folder and locate the 'index.html' file.
3. Double-click on the 'index.html' file to open it in your default web browser.

Please note that an internet connection is not required to view the website, as it is a static webpage.
